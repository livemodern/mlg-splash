# mlg-splash

Splash page / launchpad for Modern Living Group internal tools at `mlrecloud.com`.

Static HTML site with no build step. Deployed to Vercel, point apex domain `mlrecloud.com` here.

## Tools linked from this splash

- **New Construction Tool** → `new.mlrecloud.com`
- **Image Prep ("Pics")** → `pics.mlrecloud.com`
- **CRM (Follow Up Boss connector)** → `crm.mlrecloud.com`
- **Registration Bot** → `bot.mlrecloud.com`

## Local preview

Just open `index.html` in any browser. No build step.

## Deploy

Project is live on Vercel under the `Modern Living Group` team. Push to `main` triggers
auto-deploy. The `mlrecloud.com` apex domain is attached to this project.
